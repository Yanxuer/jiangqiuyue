##  图片规格

### 基本规格
| 项目 | 要求 |
|------|------|
| **画布尺寸** | 120×160 像素 |
| **格式** | PNG (透明背景) |
| **色彩** | 32位 (RGBA) |
| **文件命名** | `0001.png` ~ `00XX.png` (4位数字，从1开始) |

### 角色设计参考
根据您提供的参考图片，角色应具备以下特征：

- **发型**: 深色长发，齐刘海
- **眼睛**: 蓝色大眼睛，带高光
- **配饰**: 白色头戴式耳机
- **服装**: 深色上衣
- **风格**: 日系二次元，像素或矢量风格均可
- **表情**: 温柔可爱，符合"江秋月"人设

### 各动画状态说明

#### idle (待机, 30帧, 循环)
- 角色自然呼吸起伏
- 头发轻微飘动
- 眨眼动画（约每3秒一次）
- 建议：第1帧和第30帧应相似，确保循环平滑

#### talk (说话, 18帧, 不循环)
- 嘴巴开合动画
- 头部微动
- 结束后自动返回 idle
- 建议：第1帧接近idle姿态，最后一帧也是

#### happy (开心, 20帧, 不循环)
- 眯眼笑
- 头部上下弹动
- 可能加上腮红效果
- 建议：表现角色开心的状态

#### thinking (思考, 40帧, 循环)
- 手托下巴或歪头
- 眼睛向上看
- 缓慢循环
- 建议：表现角色正在思考的状态

#### click (点击反应, 15帧, 不循环)
- 被点击时的弹跳反应
- 表情惊讶或开心
- 建议：快速、有趣的反应

#### wave (挥手, 20帧, 不循环)
- 手臂挥动
- 微笑
- 双击触发的打招呼动画

##  制作建议

### 工具推荐
1. **Aseprite** - 专业的像素画工具 ($19.99)
2. **Photoshop** - 标准图像处理 ($20.99/月)
3. **Procreate** - iPad绘图 ($12.99)
4. **GIMP** - 免费开源 (推荐预算有限)
5. **Piskel** - 免费在线像素画工具

### 制作流程
1. 创建 120×160 像素的画布
2. 绘制角色基础姿态（第1帧）
3. 复制图层，制作动画关键帧
4. 调整细节，确保动作流畅
5. 导出为PNG序列（透明背景）
6. 按 `0001.png` ~ `00XX.png` 命名

### 参考风格
- **Shimeji** - 桌面宠物风格
- **Kikoeru** - 二次元桌面助手
- **Desktop Mate** - 虚拟助手角色
- **Live2D** - 可参考其2D角色设计

### 透明度处理
- 角色区域：不透明
- 背景区域：完全透明 (Alpha=0)
- 边缘：可以适当羽化使过渡更自然

## ⚠️ 注意事项

1. **尺寸一致性** - 所有帧必须是 120×160 像素
2. **命名规范** - 必须使用4位数字补零
3. **透明背景** - 必须保存为PNG且带Alpha通道
4. **文件数量** - 确保每组的帧数与代码配置一致
5. **版权** - 确保使用的素材有合法使用权
